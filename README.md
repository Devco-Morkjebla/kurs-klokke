# kurs-klokke


1. Start med å åpne VScode eller en annen editor.

Skriv så inn taggene for å opprette en nettside. 

Pass på at du får med `<html>`, `<body>`, `<head>`, `<style>`, og `<script>` taggen

Filen burde da se noenlunde sånn ut:
```html
<html lang="en">

<head>
  
</head>

<body>

</body>

</html>

<style>

</style>

<script>
    
</script>

```
